# StudyM8 📚

A mobile-first Progressive Web App (PWA) for South African Grade 12 learners to track their assessments, compete with friends, and stay motivated throughout matric.

## Features

- **Assessment Planner** - All assessments across 7 subjects with due dates, weights, and marks
- **Countdown & Alerts** - Countdown to next assessment, urgent/in-progress status badges
- **Calendar View** - Month view with color-coded dots per subject
- **Profile** - Upload photo, set name, school, motto, and select your subjects
- **Social / Squad** - Share progress via WhatsApp, add friends, compete on a leaderboard
- **Gamification** - XP system, 8 levels, 12 achievements, daily streak tracker, confetti animations
- **Offline Ready** - Works without internet via service worker caching
- **Installable** - Add to home screen on any phone for native app experience

## Subjects Supported

- Afrikaans Huistaal
- English First Additional Language
- Geografie
- Landboubestuurspraktyke
- Lewensoriëntering
- Lewenswetenskappe
- Wiskundige Geletterdheid

## Install on Phone

1. Visit the GitHub Pages URL in Chrome (Android) or Safari (iPhone)
2. Tap **"Add to Home Screen"**
3. Opens like a native app

## Tech

Single HTML file PWA - no build tools, no frameworks, no backend. Pure HTML/CSS/JS with localStorage for persistence.
